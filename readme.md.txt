Hospital Management System (Java)

Welcome to the Hospital Management System project! This project is developed using the Java programming language and allows users to manage hospital operations efficiently. The system provides a user-friendly interface for patient registration, appointment scheduling, and medical record management.

Features

Patient Registration: Users can register new patients by providing necessary details such as name, age, gender, and contact information.

Appointment Scheduling: The system allows scheduling appointments with doctors based on availability.

Medical Records Management: Maintains a record of patient diagnoses, prescriptions, and treatment history.

Billing System: Generates and manages hospital bills for patients based on consultations, treatments, and services availed.

Doctor & Staff Management: Allows adding, updating, and managing doctors and hospital staff details.

Installation

To run the Hospital Management System, follow these steps:

Clone or download the project files from the GitHub repository: Hospital-Management-in-Java.

Compile the source code files using the Java compiler.

javac HospitalManagementSystem.java

Execute the compiled Java program.

java HospitalManagementSystem

Usage

Upon running the Hospital Management System, you will be presented with a menu-driven interface. Follow the on-screen instructions to perform various operations, such as patient registration, appointment scheduling, and medical record updates. The system will display relevant information and prompts to guide you through the process. Manage hospital operations efficiently!
